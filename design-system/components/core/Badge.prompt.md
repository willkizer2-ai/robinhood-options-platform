A compact uppercase label for strategy tags, statuses, and categories — use it wherever a short piece of metadata needs emphasis.

```jsx
<Badge tone="gold" variant="soft">Caution</Badge>
<Badge tone="up" variant="solid">CALL</Badge>
<Badge tone="accent" variant="outline">ICT V4.1</Badge>
```

Tones: `neutral` / `accent` / `up` (teal) / `down` (magenta) / `gold`. Variants: `soft` (default), `solid`, `outline`. Sizes `sm`/`md`. Pass `icon` for a leading glyph.
