# Marketing Landing — UI kit

- A SPINX-style dark marketing landing page for **Web Trace Portfolio Management** —
sticky nav, a large gradient hero, a trusted-by strip, a
six-card feature grid, a stat band, a closing CTA, and a footer.

## Run
Open `index.html` (single self-contained file; loads the bundle + Lucide inline).

## Layout (SPINX reference, Web Trace identity)
- **Nav** — logo lockup, center links, sign-in + "Open the desk".
- **Hero** — eyebrow, oversized headline with a periwinkle gradient accent word,
  subcopy, and dual CTAs.
- **Strip** — mono capability tags.
- **Features** — 3×2 grid of capability cards with Lucide icons.
- **Stat band** — backtested edge in oversized Eagle Lake numerals.
- **Final CTA** + **footer** with the not-financial-advice disclaimer.

## Components used
`Button`, `Badge`, `StatusPill` from the bundle; everything else is page-specific
layout styled entirely with design-system tokens.
