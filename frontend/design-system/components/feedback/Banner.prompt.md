A status strip for the headline state of a card or section — DO TAKE, Stop Loss Hit, awaiting entry, info notes.

```jsx
<Banner tone="success" solid>DO TAKE</Banner>
<Banner tone="danger">Stop Loss Hit</Banner>
<Banner tone="neutral" align="left" icon={<Icon name="clock" />}>Awaiting entry level</Banner>
```

Tones: info / success / danger / warn / neutral. `solid` fills the background; default is a soft tint.
